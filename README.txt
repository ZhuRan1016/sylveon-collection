SYLDB Web V2
================

直接上传本文件夹内所有文件到 GitHub 仓库根目录：
- index.html
- style.css
- app.js
- master.json
- manifest.json
- service-worker.js

功能：
- 43项日版仙子伊布收藏数据库
- 收藏状态自动保存在当前浏览器
- 搜索、时代筛选、状态筛选、稀有度筛选
- 总进度与各时代进度
- 导出/导入备份
- 迁移旧版收藏记录
- PWA与离线缓存

说明：
- 当前数据库为v1.0暂定数据接入版。
- 暂未内置官方卡图，因此使用稳定占位图；以后只需补image_url字段即可显示图片。
